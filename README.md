## vxnet

Official website made for vxnet, available [here](https://simswapp.ing)

### Contribute

To setup the repo:

```
git clone https://github.com/vxnetsh/website
cd website
bun i
```

And to run in dev nextjs dev mode:

```
bun dev
```
// 
```
npm run build
npm start
```
